Collectors Curator

Collectors Curator is a C# WinForms application backed by a MySQL database that helps users organize and analyze personal collections by providing structured item storage, detailed attributes, and fast searching tools. It allows collectors to add, edit, and categorize items with fields like name, condition, quantity, value, and notes, while offering filters, sorting, and keyword search for navigating large collections. The system includes predictive logic to highlight missing items, duplicates, and value trends, giving users insight into the completeness and worth of their collection. Built with a relational database and a clean, user friendly interface. This project also features standard login with the ability to create an account, login with a username and password, fully hashed passwords for security on the backend and a separation between standard user privileges and administrative privleges.

Life Point Counter

This program provides a complete system for recording, normalizing, and analyzing match data for a competitive card game. It includes a match writer that logs results in a consistent format and a reader that cleans and normalizes the data by trimming whitespace, fixing casing, and resolving player and deck names. This prevents duplicates and parsing errors while keeping the match history accurate. As the program runs, it maintains a live action log, displays the most recent action, and supports prepared actions for quick input. All duels are written to a text file, which is later parsed to generate detailed reports on match results. The stats engine tracks wins, losses, opponents, and deck usage, calculates raw win rates, and applies Bayesian smoothing and matchup‑adjusted scoring. Decks are ranked by their final adjusted score and assigned a tier (S through F) based on their relative standing in the meta. The result is a clean, reliable snapshot of player performance, deck strength, and overall game balance, supported by a fully normalized history and real‑time action tracking. NOTE: The version of this available in the repo is not up to date and doesn't have all features of the current version. Will update when I complete the finalized version.

SQL Database with PHP Forms

I built a PHP and MySQL database project using XAMPP to run Apache and MySQL locally. I created my database in phpMyAdmin, wrote the SQL for the tables, and then connected everything through PHP using mysqli. I kept the connection code in its own file and wrote separate scripts to create tables, insert sample data, and display results in the browser. I used css to give the forms a cleaner, more modern look. Most of the project was about getting comfortable with database structure, testing CRUD operations, and making sure my PHP pages could reliably read and write data. I also exported the full SQL file through phpMyAdmin and verified everything by checking the tables directly.

Python Population Comparison Game

This was one of my earlier projects for a python class I took in college. Unfortuantely I don't have most of the files for this project and the few files from this class that I still have aren't really notable enough to share, so I have a video where I show my code and describe the function of the project. For this project I created a game in which the user will be asked which of two randomly selected countries (and some territories) has a larger population. Once they give an answer they are told whether they were right or wrong and a bar graph will pop up to show the difference in population between the two countries. You earn one point for every right answer and at the end of the game the program saves your score if it's a high score. I found a CSV online with data on all the countries in the world and their respective populations (this data would be outdated by now) For this project I used the random, json, csv and matplotlib/pyplot libraries.
