Collectors Curator

Collectors Curator is a C# WinForms application backed by a MySQL database that helps users organize and analyze personal collections by providing structured item storage, detailed attributes, and fast searching tools. It allows collectors to add, edit, and categorize items with fields like name, condition, quantity, value, and notes, while offering filters, sorting, and keyword search for navigating large collections. The system includes predictive logic to highlight missing items, duplicates, and value trends, giving users insight into the completeness and worth of their collection. Built with a relational database and a clean, user friendly interface. This project also features standard login with the ability to create an account, login with a username and password, fully hashed passwords for security on the backend and a separation between standard user privileges and administrative privleges.

Life Point Counter

This program provides a complete system for recording, normalizing, and analyzing match data for a competitive card game. It includes a match writer that logs results in a consistent format and a reader that cleans and normalizes the data by trimming whitespace, fixing casing, and resolving player and deck names. This prevents duplicates and parsing errors while keeping the match history accurate. As the program runs, it maintains a live action log, displays the most recent action, and supports prepared actions for quick input. All duels are written to a text file, which is later parsed to generate detailed reports on match results. The stats engine tracks wins, losses, opponents, and deck usage, calculates raw win rates, and applies Bayesian smoothing and matchup‑adjusted scoring. Decks are ranked by their final adjusted score and assigned a tier (S through F) based on their relative standing in the meta. The result is a clean, reliable snapshot of player performance, deck strength, and overall game balance, supported by a fully normalized history and real‑time action tracking.
